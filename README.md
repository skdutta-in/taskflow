# TaskFlow

A mobile-first task manager with dependency tracking, priority levels, backlog detection, and CSV export/import.

## Features
- ✅ Dependent tasks (Depends On + It Blocks)
- ✅ 5 priority levels (Critical, Urgent, High, Normal, Low)
- ✅ Backlog detection — overdue tasks shown first in Today
- ✅ Today / Tomorrow / This Week / All Tasks views
- ✅ CSV Export & Import
- ✅ PWA — works offline, installable on iPhone & Android

## Local Development

```bash
npm install
npm run dev
```

## Build for Production

```bash
npm run build
```

## Deploy
Connect this repo to [Vercel](https://vercel.com) for automatic deployments.
